# 🚗 VehicleGuard — Intelligent Vehicle Safety & Theft Prevention System

</summary>

## Overview

VehicleGuard is a smart IoT-powered vehicle monitoring system that combines embedded hardware, cloud services, and web technologies to improve vehicle safety, accident monitoring, and theft prevention.

The system continuously collects sensor data, processes it in real time, stores it on Firebase, and displays it through a responsive web dashboard.

### Project Highlights

| Category | Description |
|-----------|-------------|
| Technology | ESP32 • Firebase • HTML • CSS • JavaScript |
| Scale | IoT Real-Time Monitoring |
| Performance | Live Sensor Updates |
| Cloud | Firebase Realtime Database |
| Security | Secure Cloud Communication |
| Impact | Smart Vehicle Safety Platform |
